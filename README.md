# Write a function to find the maximum and minimum values in a dictionary

Define a function that takes a dictionary as a parameter and returns a tuple of the maximum and minimum value.

_Example:_

**Input:**

`{'a': 1, 'b': 2, 'c': 3}`

<br>

**Output:**

 
`(3, 1)`

_Challenge:_

Do not use for-loop.
